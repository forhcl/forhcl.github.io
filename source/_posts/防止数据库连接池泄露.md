Web应用程序必须显式关闭ResultSet、Statement和Connection。

```java
 public static void free(ResultSet resultSet,Statement statemen,Connection connection){   
		try {
            if(resultSet != null)
                resultSet.close();
        } catch (SQLException e) {
            e.printStackTrace();
        }finally{
            if(statemen != null)
                try {
                    statemen.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }finally{
                    if(connection != null)
                        try {
                            connection.close();
                        } catch (SQLException e) {
                            e.printStackTrace();
                        }
                }
        }
}
```
